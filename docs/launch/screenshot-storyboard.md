Tell a 6-frame story: Promise → Loop → Progress → Status → Consistency → Differentiator

Each screenshot should work standalone AND as part of a sequence. Dark premium aesthetic throughout.

## Screenshot Production Notes

### Technical Requirements
- iOS: 6.7" (1290 × 2796 px), 6.5" (1284 × 2778 px), 5.5" (1242 × 2208 px)
- Android: Feature graphic (1024 × 500), Screenshots (min 320px, max 3840px)
- All screenshots should use the app's actual dark premium design system
- No device frames needed for store submission (stores add them)

### Content Requirements
- Use realistic mission titles (e.g., "Complete 30-min deep work session," "Review trading journal")
- Show realistic stats (Level 7, 1,250 XP, 3 skills at Green/Blue)
- Use the actual app color palette (dark backgrounds, discipline purple accents)
- All text must be legible at store thumbnail size
