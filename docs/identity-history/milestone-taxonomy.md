# Identity History — Milestone Taxonomy

## History Types (5)

| Type         | Purpose                              | Example Subtypes                          |
| ------------ | ------------------------------------ | ----------------------------------------- |
| first        | First time user does something       | first_mission, first_proof_approved        |
| growth       | Level, skill, prestige progression   | level_milestone, skill_rank_up             |
| status       | Identity/status item acquisition     | status_item_acquired, room_transformation  |
| recovery     | Coming back from setbacks            | comeback_return, streak_recovered          |
| consistency  | Sustained discipline over time       | consistency_7d, proof_quality_streak       |

## History Subtypes (27)

### Firsts (11)
- `first_mission` — First mission created
- `first_proof_approved` — First proof approved by judge
- `first_reward` — First coins earned
- `first_streak_3` — First 3-day streak
- `first_streak_7` — First 7-day streak
- `first_purchase` — First marketplace purchase
- `first_equip` — First item equipped
- `first_room_item` — First room upgrade
- `first_car` — First car unlocked
- `first_visual_change` — First visual evolution
- `first_comeback` — First comeback after inactivity

### Growth (6)
- `level_milestone` — Level 5, 10, 15, 20, 25, 30, 40, 50, 75, 100
- `skill_rank_up` — Skill advances to new rank (gray→green→blue→purple→gold→red)
- `streak_milestone` — Streak reaches 3, 7, 14, 30, 60, 100 days
- `trust_improvement` — Trust score improved significantly
- `arc_transition` — Narrative arc changed
- `prestige_milestone` — Prestige tier reached

### Status (4)
- `status_item_acquired` — Status item obtained
- `room_transformation` — Room upgraded
- `wardrobe_set` — Complete wardrobe set equipped
- `car_upgrade` — Car upgraded

### Recovery (4)
- `comeback_return` — Returned after 3+ days of inactivity
- `streak_recovered` — Streak rebuilt after loss
- `quality_improvement` — Proof quality improved
- `momentum_rebuilt` — Momentum rebuilt after slow period

### Consistency (6)
- `consistency_3d` — 3 consecutive days of activity
- `consistency_7d` — 7 consecutive days
- `consistency_14d` — 14 consecutive days
- `consistency_30d` — 30 consecutive days
- `proof_quality_streak` — 5+ consecutive high-quality proofs
- `trust_stable` — Trust score stable for 14+ days

## Importance Levels

| Level       | Memory Bucket | Decay          | Examples                                |
| ----------- | ------------- | -------------- | --------------------------------------- |
| iconic      | permanent     | Never          | first_mission, first_purchase, consistency_30d |
| major       | permanent     | Never          | level_milestone, comeback_return, first_streak_7 |
| meaningful  | long_term     | After 30 days  | status_item_acquired, consistency_7d    |
| contextual  | recent        | After 7 days   | trust_improvement, consistency_3d       |
