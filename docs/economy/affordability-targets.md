# Affordability Targets — v1.0

Source of truth: `artifacts/api-server/src/lib/economy/economyConfig.ts`

## User Archetypes

| Archetype | Missions/Day | Avg Reward/Mission | Daily Bonuses | Total/Day |
|-----------|-------------|-------------------|---------------|-----------|
| Casual | 2-3 | ~30c | ~10c | ~40-100c |
| Engaged | 4-6 | ~50c | ~20c | ~120-300c |
| Power | 7-10 | ~60c | ~40c | ~300-600c |

## Day 1 — New User

| Question | Answer |
|----------|--------|
| First meaningful purchase? | Focus Trophy (80c) — achievable in 2-3 missions |
| First wearable? | Starter Timepiece (200c) — Day 1 for engaged, Day 2 for casual |
| First room item? | Indoor Plant Collection (300c) — Day 2-3 for casual |
| How many approved missions needed? | 2-5 missions for first purchase |
| Emotional goal | "I earned something real. I own something." |
| Risk if broken | User sees nothing attainable → churns before Day 2 |

## Day 7 — Engaged User

| Question | Answer |
|----------|--------|
| Cumulative earnings (engaged)? | ~700-2100c |
| Wardrobe upgrades? | 1-3 items (Hoodie 800c, Chrono 900c, Bifold 600c) |
| Room upgrades? | 1-2 items (Oak Desk 800c, LED Lighting 900c, Dark Theme 1000c) |
| First car? | Starter Ride (500c) — achievable Day 3-5 for engaged user |
| Status items? | Focus Trophy, maybe Discipline Medal |
| Aspiration targets visible? | Series M car (2500c), Royal watch (3500c) |
| Risk if broken | Stagnation → "I earn but can't buy anything I want" |

## Day 30 — Active User

| Question | Answer |
|----------|--------|
| Cumulative earnings (engaged)? | ~3600-9000c |
| Car progression? | Starter Ride owned, possibly Series M or Alpine GT in reach |
| Wardrobe? | 4-6 items including some rare/uncommon pieces |
| Room? | Dark Office environment, 3-5 decor items |
| Total completion? | <30% of total catalog |
| Aspirational goals remaining? | Continental S, Phantom Noir, Genève Perpetual, Executive Suite |
| Risk if broken (too generous) | Power users trivialize prestige items |
| Risk if broken (too stingy) | Casual users stuck with starter gear for weeks |

## Day 90 — Dedicated User

| Question | Answer |
|----------|--------|
| Cumulative earnings (engaged)? | ~10,800-27,000c |

## Critical Balance Questions

| Question | Current Answer | Assessment |
|----------|---------------|------------|
| How fast does a new user earn their first non-free wearable? | 2-5 missions (~Day 1-2) | ✅ Good |
| How long should the first car take? | ~5-7 days for engaged user | ✅ Good |
| How long should a premium room take? | ~2-3 weeks for engaged user | ✅ Good |
| Can a highly active user buy too many rooms in one week? | Could buy 3-4 basic items — acceptable | ✅ OK |
| Can a casual user buy nothing meaningful for too long? | First purchase at 80c reachable Day 1 | ✅ Good |
| Can a power user trivialize the economy in 30 days? | ~8000c = 1 mid-car + several wearables, not everything | ✅ Balanced |
| Are legendary items protected? | Carbon RM (18k) + Vulcan R (25k) require months | ✅ Protected |
