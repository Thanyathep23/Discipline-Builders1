PART 1 — VOXEL CHARACTER ARCHITECTURE
==================================================

Build the voxel character using React Native View
components arranged in a grid/block system.

Each "voxel" = a styled View with:
- Fixed pixel size (8x8 or 6x6 points)
- Background color
- Subtle border for depth effect

CHARACTER CANVAS:
- Total size: 300x500 points on screen
- Voxel size: 8x8 points
- Grid: approximately 37 x 62 voxels
- Background: transparent (dark bg from screen)

VOXEL RENDERING ENGINE:
Create a VoxelGrid component that accepts
a 2D array of color values and renders them:

type VoxelMap = (string | null)[][]

const VoxelGrid = ({ 
  map, 
  voxelSize = 8 
}: { 
  map: VoxelMap
  voxelSize: number 
}) => {
  return (
    <View style={{ flexDirection: 'column' }}>
      {map.map((row, y) => (
        <View key={y} style={{ flexDirection: 'row' }}>
          {row.map((color, x) => (
            <View
              key={x}
              style={{
                width: voxelSize,
                height: voxelSize,
                backgroundColor: color || 'transparent',
                borderRightWidth: color ? 0.5 : 0,
                borderBottomWidth: color ? 0.5 : 0,
                borderColor: color ? 
                  darken(color, 0.15) : 'transparent',
              }}
            />
          ))}
        </View>
      ))}
    </View>
  )
}

PART 5 — ROTATION / VIEW SYSTEM
==================================================

Implement swipe-to-rotate between 3 views:

5A. Gesture Handler (no native modules needed):

Use PanResponder (built into React Native):

const panResponder = PanResponder.create({
  onStartShouldSetPanResponder: () => true,
  onPanResponderMove: (evt, gestureState) => {
    const dragPercent = gestureState.dx / SCREEN_WIDTH
    const newAngle = 
      (currentAngle + dragPercent * 360 + 360) % 360
    setRotationAngle(newAngle)
  },
  onPanResponderRelease: (evt, gestureState) => {
    snapToNearestView(currentAngle)
  },
})

5B. View Selection Logic:

angle 0-45°: FRONT view
angle 45-135°: SIDE view  
angle 135-225°: BACK view
angle 225-315°: SIDE view (mirrored)
angle 315-360°: FRONT view

5C. Crossfade Transition:

Use React Native Animated.Value for opacity:
When switching views:
  Fade out current: opacity 1→0 (150ms)
  Fade in next: opacity 0→1 (150ms)
  Total transition: 300ms smooth

5D. Rotation Indicator:

[●] [○] [○] [○]

PART 6 — VOXEL DEPTH EFFECT
==================================================

Make voxels look 3D using border shading:

Each filled voxel View has:
- topBorder: color lightened 20% (highlight)
- leftBorder: color lightened 10%  
- rightBorder: color darkened 15% (shadow)
- bottomBorder: color darkened 20% (deep shadow)

This creates isometric voxel depth feel
matching the reference image style.

Helper function:
const getVoxelStyle = (baseColor: string) => ({
  backgroundColor: baseColor,
  borderTopColor: lighten(baseColor, 0.2),
  borderLeftColor: lighten(baseColor, 0.1),
  borderRightColor: darken(baseColor, 0.15),
  borderBottomColor: darken(baseColor, 0.2),

PART 10 — PERFORMANCE
==================================================

Voxel optimization for 60fps mobile:

1. Use useMemo for voxel map calculations
   Recalculate only when props change

2. FlatList instead of map() for large grids
   Use getItemLayout for optimization

3. Pre-calculate all voxel maps at init
   Store as constants, not computed each render

4. Consider using a Canvas/Skia approach
   if View-based voxels are too slow:
   
   import { Canvas, Rect } from '@shopify/react-native-skia'
   
   Draw each voxel as a Rect:
   Rect x={x*size} y={y*size} 
        width={size} height={size} 
        color={color}
   
   @shopify/react-native-skia is already
   in many Expo projects and works in Expo Go.
   Check if installed before using.

5. If neither approach is fast enough:
   Generate voxel character as SVG path
   (each voxel = rect element in SVG)
   Use react-native-svg which is already installed.

RECOMMENDED: SVG-based voxel renderer
(most compatible, already installed):

const VoxelSVG = ({ map, voxelSize }) => (
  <Svg width={map[0].length * voxelSize} 
       height={map.length * voxelSize}>
    {map.flatMap((row, y) =>
      row.map((color, x) =>
        color ? (
          <Rect
            key={`${x}-${y}`}
            x={x * voxelSize}
            y={y * voxelSize}
            width={voxelSize - 0.5}
            height={voxelSize - 0.5}
            fill={color}
            stroke={darken(color, 0.2)}
            strokeWidth={0.5}
          />
        ) : null
      )
    )}
  </Svg>
)

This uses react-native-svg (already installed)
and requires ZERO new native modules.
