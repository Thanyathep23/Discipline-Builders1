## Technical implementation

### Option A — Simplest (model-viewer web component)
Add to HTML head:
```html
<script type="module" src="https://ajax.googleapis.com/ajax/libs/model-viewer/3.5.0/model-viewer.min.js"></script>
```

Use in the character screen:
```html
<model-viewer
  src="/models/Superhero_Male_FullBody.gltf"
  camera-controls
  auto-rotate
  auto-rotate-delay="3000"
  rotation-per-second="6deg"
  camera-orbit="0deg 82deg auto"
  field-of-view="36deg"
  environment-image="neutral"
  shadow-intensity="1.2"
  exposure="1.1"
  style="width: 100%; height: 100%; background: transparent; --poster-color: transparent;">
</model-viewer>
```

### Option B — Three.js (for full control)
Install: `npm install three @types/three`

Then use THREE.GLTFLoader to load `/models/Superhero_Male_FullBody.gltf`

Lighting setup (IMPORTANT for cinematic look):
- Key light: DirectionalLight(0xFFF2D0, 3.0) at position (-1.8, 2.8, 2.0) with shadows
- Rim light: DirectionalLight(0x6AADFF, 2.5) at position (2.5, 2.0, -1.5)
- Fill light: DirectionalLight(0xC9A84C, 0.8) at position (0, 0.5, 3.0)
- Ambient: AmbientLight(0x101520, 1.5)
- Bounce: PointLight(0x9B7030, 1.0) at position (0, -0.2, 1.2)

Camera setup:
- PerspectiveCamera(36, aspect, 0.01, 100)
- Position at (0, focusY + 0.22, 2.1) where focusY = modelHeight * 0.70
- LookAt (0, focusY - 0.10, 0)

Model setup after loading:
- Scale model to 1.8m height: scale = 1.8 / boundingBoxHeight
- Move to y=0 (feet on ground)
- Enable castShadow and receiveShadow on all meshes
- Renderer: ACESFilmicToneMapping, exposure 1.1, sRGBEncoding output

## If using React / TypeScript component

Create file: `src/components/CharacterViewer.tsx`
```tsx
import { useEffect, useRef } from 'react'

// Declare model-viewer as a custom element for TypeScript
declare global {
  namespace JSX {
    interface IntrinsicElements {
      'model-viewer': React.DetailedHTMLProps<React.HTMLAttributes<HTMLElement> & {
        src?: string
        alt?: string
        'camera-controls'?: boolean
        'auto-rotate'?: boolean
        'auto-rotate-delay'?: string
        'rotation-per-second'?: string
        'camera-orbit'?: string
        'field-of-view'?: string
        'environment-image'?: string
        'shadow-intensity'?: string
        'exposure'?: string
      }, HTMLElement>
    }
  }
}

export default function CharacterViewer() {
  useEffect(() => {
    // Load model-viewer script once
    if (!document.querySelector('script[data-mv]')) {
      const s = document.createElement('script')
      s.type = 'module'
      s.src = 'https://ajax.googleapis.com/ajax/libs/model-viewer/3.5.0/model-viewer.min.js'
      s.setAttribute('data-mv', '1')
      document.head.appendChild(s)
    }
  }, [])

  return (
    <model-viewer
      src="/models/Superhero_Male_FullBody.gltf"
      camera-controls
      auto-rotate
      auto-rotate-delay="3000"
      rotation-per-second="6deg"
      camera-orbit="0deg 82deg auto"
      field-of-view="36deg"
      environment-image="neutral"
      shadow-intensity="1.2"
      exposure="1.1"
      style={{ width: '100%', height: '100%', background: 'transparent' }}
    />
  )
}
```
