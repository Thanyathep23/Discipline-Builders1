==================================================
PART 4 — 360° ROTATION SYSTEM
==================================================

Implement smooth 360° rotation:

4A. Gesture Handler

Use react-native-gesture-handler PanGestureHandler:

const panGesture = Gesture.Pan()
  .onUpdate((e) => {
    rotation.value = startRotation + 
      (e.translationX / containerWidth) * Math.PI * 2
  })

Map horizontal drag to Y-axis rotation.
Full swipe across screen = one full rotation.

4B. Rotation Behavior
- Start angle: front facing (0°)
- Drag right: rotates to show left side → back
- Drag left: rotates to show right side → back
- Momentum: slight deceleration after release
- Snap to: front (0°), side (90°), 
           back (180°), other side (270°)
           if user releases near those angles

4C. Auto-rotate (Optional)
When no interaction for 3 seconds:
Slowly auto-rotate 360° once to "show off"
Then stop at front view.

4D. Camera Position
Camera: position [0, 5, 12]
Looking at: [0, 4, 0] (character center)
FOV: 40 (not too wide — prevents distortion)
Near/Far: 0.1 / 100
