2A. Room Canvas Component

Create: artifacts/mobile/components/room/RoomCanvas.tsx

Props:
- equippedItems: RoomItem[] (all currently placed items)
- showCharacter: boolean
- characterState: CharacterVisualState (from existing system)
- roomTheme: string
- onZoneTap: (zone: string) => void

The room canvas renders:
- background based on room theme
- each equipped decoration in its zone
- character overlay if showCharacter is true
- tap zones for placing/replacing items
- ambient lighting effect if lighting item equipped

Use SVG or React Native View composition.
Each zone is a positioned absolute layer.
Character appears as the existing character SVG/component.

Rewrite the Room screen to support full decoration:

3A. Screen Layout

HEADER:

- Base score progress bar

- Each shows item icon, name, slot
- "Active" chip if currently placed
- Tap to manage (replace/remove)

SHOP SECTION:
- "Decorate Your Space" header
- 3 featured items (recommended for current level)
- "Browse All Room Items →" button

3B. Zone Tap Behavior

When user taps an empty zone in the room canvas:
- show a bottom sheet: "Place Item Here"
- filter shop items to that zone category
- show owned-but-not-placed items first
- then show locked/purchasable items

When user taps an occupied zone:
- show options: "Replace" / "Remove"
- Replace → same filter sheet
- Remove → confirm → item goes back to inventory

3C. Purchase Flow

When user taps a room item to buy:
- show item detail bottom sheet
- show room preview: "How this will look in your room"
  (small preview with item placed in correct zone)
- "Purchase for X Coins" button
- on purchase: item added to inventory + placed in room
- room canvas updates immediately

==================================================
PART 5 — ROOM PROGRESSION SYSTEM
==================================================

5A. Base Score Calculation

Room base score = sum of:
- placed item rarity weights:
  Common: +3
  Refined: +8
  Prestige: +15
  Elite: +25
  Legendary: +40
- room theme equipped: +20
- character in room: +5
- lighting equipped: +8

5B. Room Tiers

Tier 0 (Unranked): 0-29 points → "Standard Base"
Tier 1 (Recruit): 30-74 → "Emerging Workspace"  
Tier 2 (Rising): 75-149 → "Professional Setup"
Tier 3 (Established): 150-249 → "Premium Command Center"
Tier 4 (Elite): 250-399 → "Executive Suite"
Tier 5 (Legendary): 400+ → "Iconic Command Center"

5C. Tier Benefits

Higher room tier:
- increases passive Finance/Lifestyle signal on Character Hub
- unlocks room-exclusive badge milestones
- shown as prestige signal on profile/showcase

5D. Milestone Badges

"First Decoration" — first room item purchased
"Workspace Ready" — desk + monitor both placed
"Coffee Culture" — coffee item placed
"Command Center I" — reach Tier 2
"Executive Setup" — reach Tier 4
"Full Setup" — all 6 zone types filled

PART 6 — BACKEND
==================================================

6A. Room State API

GET /api/room/state
Returns:
- placed items per zone
- room tier + score
- isCharacterInRoom
- milestone badges earned

POST /api/room/place
Body: { itemId, zone }
- validates user owns item
- places item in zone (replaces existing)
- recalculates room score
- checks for new milestone badges
- returns updated room state

POST /api/room/remove  
Body: { itemId }
- removes item from zone
- recalculates score

POST /api/room/toggle-character
Body: { inRoom: boolean }
- saves character-in-room preference

6B. Schema

Add to user state or create room_state table:
- user_id
- placed_items: JSON { zone: itemId }
- is_character_in_room: boolean
- room_score: integer
- room_tier: integer
- updated_at
