==================================================

1A. Complete Visual Overhaul

The room canvas needs to feel like a REAL space.
Currently it looks like a UI diagram with icons.

Implement this exact layering system:

LAYER 1 — Room Shell (bottom)
Background: linear gradient top-to-bottom
Top (wall): #0D0E1A
Bottom (floor): #080910
Border radius: 16px

LAYER 2 — Floor Surface
Position: bottom 45% of canvas
Background: linear gradient
#111220 → #0C0D18
Add perspective lines using SVG:

5 lines converging to vanishing point
Color: rgba(255,255,255,0.035)
Creates real depth illusion


LAYER 3 — Floor Glow (ambient light pool)
Shape: ellipse, 70% width, 25% height
Position: center, 65% from top
Background: radial gradient
rgba(109,40,217,0.18) center → transparent edge
Blur: 20px equivalent

LAYER 4 — Zone Items (positioned absolutely)
Each zone is a View positioned in perspective:
Back zones (Lighting, Theme): y=15%, smaller scale
Mid zones (Shelf, Monitor, Trophy): y=32%, normal
Floor zones (Plants, Desk, Coffee, Audio): y=52%, larger
Character: y=60%, front-center

LAYER 5 — Foreground vignette
Subtle dark edges left/right
gradient: transparent center → rgba(0,0,0,0.3) edges
Makes room feel enclosed

1B. Perspective Scaling

Items further back appear smaller:
Back row scale: 0.75
Mid row scale: 0.88  
Front row scale: 1.0
Character scale: 1.0

This single change makes the room feel 3D.

1D. Character Grounding (CRITICAL FIX)
