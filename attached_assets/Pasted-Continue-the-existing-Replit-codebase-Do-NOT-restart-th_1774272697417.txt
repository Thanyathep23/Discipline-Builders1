PART 1 — 7-LAYER CHARACTER RENDERER
==================================================

The character SVG must be composed of these 7 layers,
each independently changeable:

LAYER 1 — BODY BASE
  Always present
  Skin tone applied from customization
  Adult proportions (not cartoon)
  Full body head-to-feet

LAYER 2 — POSTURE LAYER
  Changes based on FITNESS dimension level:
  
  Fitness level 1-3 (posture: 'neutral'):
    Slightly relaxed stance
    Shoulders at natural position
    
  Fitness level 4-6 (posture: 'upright'):
    Straight spine
    Chest slightly forward
    Confident neutral
    
  Fitness level 7-8 (posture: 'athletic'):
    Broader shoulder position
    More open chest
    Energetic stance
    
  Fitness level 9-10 (posture: 'peak'):
    Commanding open stance
    Strong confident presence

  Implementation:
    4 SVG path variants for body/shoulder position
    Cross-fade between them (600ms) on level change

LAYER 3 — OUTFIT LAYER
  Changes based on FINANCE dimension level
  AND equipped wardrobe items:
  
  Finance level 1-3:
    Default starter outfit
    White shirt, dark jeans
    
  Finance level 4-6:
    Slightly better cut
    Same colors, improved fabric detail
    
  Finance level 7-8:
    Premium outfit quality
    Refined collar, better fit
    
  Finance level 9-10:
    Elite outfit quality
    Luxury fabric signals visible
    
  Rule:
    IF wardrobe item equipped → use wardrobe item
    ELSE → use finance tier default
    
  Wardrobe override props:
    equippedTopStyle: string | null
    equippedBottomStyle: string | null

LAYER 4 — WATCH LAYER
  Only renders if watch is equipped
  Position: left wrist
  Small but clearly visible
  Props: equippedWatchStyle: string | null
  
  Watch styles:
    null → no watch
    'basic_watch' → simple circle on wrist
    'sport_watch' → sport styling
    'premium_watch' → refined dial visible
    'elite_watch' → luxury crown detail

LAYER 5 — ACCESSORY LAYER
  Renders if accessory equipped
  Ring on right hand or pocket square on chest
  Props: equippedAccessoryStyle: string | null

LAYER 6 — PRESTIGE OVERLAY
  Changes based on PRESTIGE dimension level:
  
  Prestige level 1-3:
    No overlay (nothing added)
    
  Prestige level 4-6:
    Subtle collar refinement
    Minor quality detail on shirt
    
  Prestige level 7-8:
    Visible quality markers
    Cufflink suggestion at wrists
    
  Prestige level 9-10:
    Elite full upgrade
    Very subtle glow/aura at shoulders
    Premium presentation

LAYER 7 — CUSTOMIZATION LAYER
  Hair style SVG path (6 options)
  Hair color fill (8 colors)
  Always applied on top
  From user customization settings

PART 2 — POSTURE STAGE IMPLEMENTATION
==================================================

In characterEngine.ts, compute postureStage
directly from Fitness dimension level:

function getPostureStage(fitnessLevel: number): PostureStage {
  if (fitnessLevel >= 9) return 'peak'
  if (fitnessLevel >= 7) return 'athletic'
  if (fitnessLevel >= 4) return 'upright'
  return 'neutral'
}

Similarly for other visual stages:

function getOutfitTier(financeLevel: number): OutfitTier {
  if (financeLevel >= 9) return 'elite'
  if (financeLevel >= 7) return 'premium'
  if (financeLevel >= 4) return 'rising'
  return 'starter'
}

function getPrestigeStage(prestigeLevel: number): PrestigeStage {
  if (prestigeLevel >= 9) return 'legendary'
  if (prestigeLevel >= 7) return 'visible'
  if (prestigeLevel >= 4) return 'subtle'
  return 'none'
}

function getDisciplineRefinement(disciplineLevel: number): RefinementStage {
  if (disciplineLevel >= 9) return 'commanding'
  if (disciplineLevel >= 7) return 'sharp'
  if (disciplineLevel >= 4) return 'composed'
  return 'casual'
}

Include all 4 stage values in CharacterVisualState:
{
  postureStage: PostureStage
  outfitTier: OutfitTier
  prestigeStage: PrestigeStage
  refinementStage: RefinementStage
  skinTone: string
  hairStyle: string
  hairColor: string
  equippedWatchStyle: string | null
  equippedTopStyle: string | null
  equippedAccessoryStyle: string | null
}
