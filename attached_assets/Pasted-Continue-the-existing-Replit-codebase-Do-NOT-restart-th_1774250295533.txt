Goal:
Make the Room screen so visually compelling that users 
immediately want to decorate it. The current version has

==================================================
PART 1 — ROOM CANVAS DEEP UPGRADE
==================================================

1A. Background & Atmosphere

Current: dark oval glow in center
Target: real room feel with depth layers

Background layers (bottom to top):

LAYER 1 — Floor
- Subtle perspective floor
- Dark charcoal with very faint grid lines
  going toward vanishing point
- Like a real room floor, not a diagram
- Color: #12131F with #1A1B2E grid lines at 8% opacity

LAYER 2 — Back Wall  
- Slightly lighter than floor
- Subtle vertical panel lines (like wall panels)
- Color: #0E0F1C
- Height: top 45% of canvas

LAYER 3 — Ambient Glow
- Soft elliptical glow on floor under items
- Not the current large oval — smaller, more realistic
- Like light pooling on the floor
- Color: accent purple at 12% opacity
- Position: center-bottom area

LAYER 4 — Ceiling Suggestion
- Very subtle darker area at very top
- Makes the space feel enclosed

1B. Zone Placement Layout

Redesign the zone positions to feel like a REAL room
not a schematic diagram:

NEW LAYOUT (perspective-aware):

BACK WALL (top area):
  [LIGHTING]     [THEME]
  Position: top-left and top-center

MID AREA:
  [SHELF]  [MONITOR]  [TROPHY]
  Position: middle row, spread across

FLOOR LEVEL:
  [CHARACTER]  [DESK]  in center
  [PLANTS] left    [COFFEE] right
  [AUDIO] far right

Items further back = slightly smaller (perspective)
Items in front = slightly larger
This creates real depth illusion

1C. Empty Zone Redesign

Current: faint icon + text label
Replace with: premium empty slot indicators

Each empty zone shows:
- Subtle dashed rounded rectangle
- Zone-specific icon centered (20px, 30% opacity)
- On press: gentle pulse animation + glow
- Color: #FFFFFF at 15% for border
- Background: item-category tint at 5% opacity

Example:
Coffee zone empty = coffee cup icon with 
warm amber tint at 5% background

1D. Placed Item Improvements

When item is placed in zone:
- Item SVG renders at proper perspective scale
- Subtle drop shadow: 0 4px 12px rgba(0,0,0,0.5)
- If item is Elite+: thin gold border glow
- If item is Legendary: subtle particle/shimmer

1E. Character Redesign in Room

Current: character floating with no grounding
Fix:
- Character stands ON the floor (proper foot position)
- Subtle shadow/reflection under feet
- Character is properly sized relative to desk
  (desk should be waist-height on character)
- Character faces slightly forward (not straight on)
- Small "YOU" label above character head
  (typography.micro, accent color, only visible briefly)

==================================================
TECHNICAL SPECS
==================================================

Canvas implementation:
- React Native View with absolute positioning
- NOT SVG viewBox for the overall canvas
  (SVGs only for individual items)
- Use Reanimated 2 for animations
- useSharedValue for smooth interpolations

Performance rules:
- Canvas re-render only when placed items change
- Memoize all item SVG components
- Avoid state updates during animations
- Target: 60fps on mid-range Android

Design tokens required:
- colors.bg.room = '#0A0B14'
- colors.bg.roomWall = '#0E0F1C'  
- colors.room.floorGrid = 'rgba(255,255,255,0.04)'
- colors.room.ambientGlow = 'rgba(139,92,246,0.12)'
- colors.room.zoneEmpty = 'rgba(255,255,255,0.12)'
- colors.room.zoneActive = accent.primary
