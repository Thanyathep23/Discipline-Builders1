PART 1 — SKILL → CHARACTER CONNECTION
==================================================

The character's appearance must be CAUSED by 
real verified behavior, not manual selection.

1A. The 4 Core Dimensions

These 4 dimensions drive character appearance:

DIMENSION 1 — FITNESS
Sources (XP gain from):
  - completed fitness missions (proof approved)
  - session duration in fitness category
  - proof quality score bonus
  - streak bonus (7+ day consistency)
  
Visual effects at different levels:
  Level 1-3: slight slouch, average posture
  Level 4-6: upright posture, better presence
  Level 7-8: confident stance, strong posture
  Level 9-10: athlete posture, peak presence

DIMENSION 2 — DISCIPLINE  
Sources:
  - mission completion rate (completed/created)
  - focus sessions without interruption
  - streak days
  - low distraction count in sessions
  
Visual effects:
  Level 1-3: casual, slightly disheveled
  Level 4-6: neat, composed appearance
  Level 7-8: sharp, refined presentation
  Level 9-10: commanding presence

DIMENSION 3 — FINANCE / LIFESTYLE
Sources:
  - finance/trading missions completed
  - equipped wardrobe item tier
  - room tier
  - total coins earned (lifetime)
  
Visual effects:
  Level 1-3: basic starter outfit
  Level 4-6: upgrading clothing quality
  Level 7-8: premium style, polished look
  Level 9-10: elite appearance, luxury signals

DIMENSION 4 — PRESTIGE
Sources:
  - overall level
  - badges/titles earned
  - missions completed total
  - skills average (all 4)
  
Visual effects:
  Level 1-3: no prestige markers
  Level 4-6: subtle quality markers
  Level 7-8: visible status indicators
  Level 9-10: elite prestige overlays

1B. Dimension XP Calculation

After every approved proof:
Calculate which dimensions get XP based on mission category:

fitness missions:
  FITNESS: +primary_xp (based on quality × duration)
  DISCIPLINE: +secondary_xp (20% of primary)

trading missions:
  FINANCE: +primary_xp
  DISCIPLINE: +secondary_xp

learning missions:
  PRESTIGE: +primary_xp
  DISCIPLINE: +secondary_xp

deep_work missions:
  PRESTIGE: +primary_xp
  FINANCE: +secondary_xp (if business related)
  DISCIPLINE: +secondary_xp

habit missions:
  DISCIPLINE: +primary_xp
  (relevant dimension): +secondary_xp

primary_xp = (quality_score × reward_coins) / 3
secondary_xp = primary_xp × 0.2

Dimension level formula:
  level = floor(sqrt(total_xp / 50)) + 1
  Max level: 10

1C. Dimension Level → Visual State

Create this lookup system:

dimensionToVisualState(dimension, level):

FITNESS visual states:
  {level: 1-3, posture: 'slouch', bodyTone: 'base', energy: 'low'}

PART 2 — CHARACTER RENDERER (LAYERED SVG)
==================================================

2A. Layer Architecture

CharacterRenderer accepts CharacterVisualState:
{
  // From Fitness
  posture: 'slouch'|'upright'|'athletic'|'peak'
  bodyTone: 'base'|'fit'|'strong'|'elite'
  
  // From Discipline
  refinement: 'casual'|'neat'|'sharp'|'elite'
  
  // From Finance/Lifestyle
  outfitTier: 'starter'|'rising'|'premium'|'elite'
  
  // From Prestige
  prestigeLevel: 0-3
  
  // Customization (from Part 3 setup)
  skinTone: string (hex)
  hairStyle: string
  hairColor: string
  
  // Equipped wardrobe
  equippedWatch: WatchStyle | null
  equippedOutfit: OutfitStyle | null
}

2B. SVG Layer Stack

Render these layers bottom to top:

LAYER 1 — BODY BASE
  Full body silhouette
  Skin tone applied
  Always present

LAYER 2 — POSTURE OVERLAY
  Subtle posture adjustment
  slouch: slight forward lean
  upright: straight neutral
  athletic: chest slightly forward, confident
  peak: commanding open stance
  
  Implemented as: different SVG path variants
  Cross-fade when tier changes

LAYER 3 — OUTFIT LAYER
  starter: white shirt + dark jeans (default)
  rising: slightly better cut, same colors
  premium: premium fabric quality visual
  elite: luxury outfit quality
  
  If equipped wardrobe item from Wardrobe system:
    Override with equipped item style
  Else: use tier-based default

LAYER 4 — WATCH LAYER
  Only renders if watch equipped
  Position: left wrist
  Small but visible
  Shows equipped watch design

LAYER 5 — PRESTIGE OVERLAY
  Level 0: nothing
  Level 1: subtle collar/detail upgrade
  Level 2: visible quality markers
  Level 3: elite full upgrade with subtle glow

LAYER 6 — CUSTOMIZATION LAYER
  Hair style applied
  Hair color applied
  (from user customization settings)

LAYER 7 — FACE LAYER
  Adult proportions (not cartoon)
  Neutral confident expression
  Reflects character energy level subtly

2C. Character SVG Quality Requirements

The character must look:
  - Adult (20-35 years old appearance)
  - Premium and aspirational
  - Entrepreneurial (not childish)
  - Clean and modern flat design
  
NOT:
  - Cartoon proportions
  - Big eyes
  - Round childish face
  - Overly simple stick figure

Full body: head to feet visible
Height ratio: approximately 1:6 (head:body)
Style: clean flat design, subtle shading
