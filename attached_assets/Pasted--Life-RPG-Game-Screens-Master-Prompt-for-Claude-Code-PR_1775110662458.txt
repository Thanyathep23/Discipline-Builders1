## TASK
Complete all game screens for the Life RPG app. The app is a gamified discipline-builder with:
- Character customization (3D model with outfit layers)
- Wardrobe system (buy/equip clothing, Prestige-gated)
- Room (Command Center) with furniture slots and tier progression
- Garage screen
- Rewards/coins system

## IMPLEMENTATION ORDER

2. Create convex/wardrobe.ts + convex/room.ts (mutations: purchaseItem, equipItem, getEquippedOutfit)
3. Create src/components/CharacterViewer.tsx (Three.js WebView, r128, OrbitControls, postMessage API)

7. Wire navigation in AppNavigator.tsx

## CHARACTER VIEWER (Three.js WebView)
- THREE version r128 (CDN compatible)
- Do NOT use CapsuleGeometry
- Three layers: Layer 0 = body, Layer 1 = underwear, Layer 2 = outfit
- React Native ↔ WebView via postMessage:
  EQUIP_ITEM { layer, meshName }
  SET_HAIR { style, color }
  SET_SKIN { color }

## WARDROBE SCHEMA
wardrobeItems: { itemId, name, category (top/bottom/shoes/accessory), price, prestigeRequired (0=all), modelLayer, thumbnailUrl, isDefault }
userWardrobe: { userId, itemId, isEquipped, purchasedAt }

## ROOM SCHEMA  
roomItems: { slot (DESK/SHELF/MONITOR/TROPHY/PLANTS/AUDIO/COFFEE), name, price, points, prestigeRequired, imageUrl }
userRoom: { userId, themeId, equippedItems [{slot, itemId}], totalPoints, currentTier }

## ROOM TIER SYSTEM
Standard Base (0-29) → Emerging Workspace (30-79) → Professional Hub (80-149) → Elite Command (150-299) → Legend HQ (300+)

## PRESTIGE GATE
const isLocked = item.prestigeRequired > userPrestige
Show lock overlay: "Unlock at Prestige {N}"

## STARTER KIT (free defaults)

- Black Boxer Briefs (layer 1)  
- No furniture (Standard Base)
