PART 2 — CAR VISUAL PRESENTATION SYSTEM
==================================================

Build a premium car visual system using SVG or high-quality
illustrated assets.

For each car, create or reference:

1. Side profile view (primary)
   - clean 3/4 or pure side view
   - dark background friendly
   - premium silhouette rendering
   - must feel like a luxury automotive ad, not a toy

2. Color variant system
   - each car must support color swapping
   - use the defined color variants per car
   - color selection visible in car detail view

3. Lock/Unlock states
   - locked: desaturated with lock icon overlay
   - unlockable: subtle accent highlight with progress indicator
   - owned: full color, owned badge
   - featured/showcased: elevated display treatment

Requirements:
- cars must look aspirational to an entrepreneur
- NOT: cartoon style, pixel art, toy-like proportions
- YES: clean, modern, premium illustrated or SVG automotive style
- if using SVG, make silhouettes accurate and proportional
- dark mode friendly
- must look good in photo mode composition with character

==================================================
PART 3 — CAR COLLECTION SCREEN UPGRADE
==================================================

Upgrade the collection screen layout:

3A. Grid layout
- 2-column card grid for collection browsing
- each card shows:
  - car silhouette/visual
  - car name
  - rarity chip
  - lock status
  - if locked: level required + coin cost
  - if owned: "Owned" chip + "Showcase" button

3B. Featured car area
- top of screen: hero display of currently featured/active car
- large visual
- car name, tier, prestige class
- "Change Featured" button
- "Photo Mode" button

3C. Filter bar
- All / Owned / Locked / By Rarity

3D. Car detail sheet
- opens as bottom sheet on card tap
- shows:
  - large car visual
  - color variant selector
  - full description
  - stats: level required, coin cost, prestige value
  - unlock progress if partially achievable
  - "Purchase" or "Locked" or "Already Owned" CTA

PART 4 — PHOTO MODE UPGRADE
==================================================

Upgrade Photo Mode to feel premium and shareable:

4A. Composition
- character on left, car behind/beside
- clean dark or gradient background options:
  - Night City (dark urban)
  - Coastal (horizon/ocean)
  - Minimal (pure dark gradient)
  - Track (subtle racing circuit feel)

4B. Overlay options
- show username + current status tier
- show car name
- toggle on/off

4C. Export
- save to camera roll / share sheet
- premium aspect ratio: 4:5 or 1:1
- subtle watermark with app name (optional toggle)

4D. Presentation
- the photo mode result should feel like a luxury lifestyle post
- NOT: gaming screenshot
- YES: premium editorial moment

Connect car ownership to the character/status system:
