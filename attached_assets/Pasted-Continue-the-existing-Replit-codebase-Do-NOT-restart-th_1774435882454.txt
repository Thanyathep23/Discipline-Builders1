==================================================
PART 2 — CHARACTER RENDERER COMPONENT
==================================================

Update CharacterRenderer to use PNG layer system:

File: artifacts/mobile/components/character/CharacterRenderer.tsx

Props remain the same:
{
  visualState: CharacterVisualState
  size: 'small'|'medium'|'large'|'full'
  bodyType: 'male'|'female'
  showShadow?: boolean
  view?: 'front'|'side' (default: 'front')
}

Render using React Native Image layers
stacked with absolute positioning:

LAYER ORDER (bottom to top):
1. Shadow/ground (ellipse blur below feet)
2. Body base (skin tone applied via tint)
3. Posture layer (changes with fitness level)
4. Bottom/jeans layer
5. Top/shirt layer  
6. Outerwear layer (if equipped)
7. Accessories layer (necklace, bracelet)
8. Watch layer (wrist, if equipped)
9. Hair layer
10. Face layer

Each layer is a PNG with transparency.
Layers stack using absolute positioning
within a fixed-size container.
