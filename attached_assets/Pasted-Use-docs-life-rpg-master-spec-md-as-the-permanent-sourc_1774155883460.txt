==================================================
A. CROSS-SYSTEM INTEGRATION PASS
==================================================

Unify the newly built game-mode systems so they feel like one connected product.

Areas to align:
- Character Status Hub language
- status tier/class language
- wearable/style language
- store/catalog/item-state language
- room/workspace terminology
- car collection terminology
- prestige/lifestyle/status presentation language
- next evolution / next unlock wording
- recommendation/guidance wording around game-mode systems
- equipped/displayed/owned/locked/premium/limited states everywhere

Requirements:
- remove conflicting labels and duplicated meanings where practical
- ensure a user can understand item state consistently across:
  - store
  - inventory
  - character
  - room
  - car collection
- ensure the game-mode expansion feels like a unified “life-status game” layer
- do not rewrite the whole IA; focus on consistency and polish

==================================================
B. GAME MODE FLOW POLISH
==================================================

Improve the high-value flows inside the new game-mode expansion.

Critical flows to polish:
1. open Game Mode -> understand current status quickly
2. see evolved character -> understand why the character looks this way
3. move from status page -> store -> purchase item -> equip/apply item
4. move from status page -> room -> apply room upgrade
5. move from status page -> car collection -> understand locked/owned/showcase state
6. use photo mode lite with owned car smoothly
7. understand what to do next after unlocking/purchasing/upgrading something

Requirements:
- reduce unnecessary taps where practical
- improve “what changed?” clarity after important actions
- improve “what should I do next?” clarity
- keep actions mobile-friendly
- ensure the user is not lost between character, room, store, and car flows

Add a targeted premium polish pass to the game-mode expansion.

Possible areas:

- smoother equipped/active/displayed feedback
- clearer visual transitions between store/inventory/character/room/car screens

==================================================

Audit and improve:

- car collection/gallery rendering

==================================================

- if small recompute/reconcile helpers are needed, keep them minimal and safe
